Due to the pandemic situation conducting an exhibition was a tedious task. To get through this problem we have created a project namely E-Exhibition.This project consists of a virtual environment with 3D models of numerous projects which solves the problem of conducting exhibition virtually.The sole reason for developing this project is to showcase many other projects made with excruciating efforts by creating their 3D Models so that it can be seen anywhere by anyone.
